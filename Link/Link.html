What are links ?

HTML links are hyperlinks. You click on the link then go to another page or document. A link does not have to be text. A link can be an image or any other HTML element The HTML "a" tag defines a hyperlink. The most important attribute of the element is the href attribute, which indicates the link's destination.

Example of link -

if you were to click on this link - Main page you would go back to the main page

Code -

To make a link you would have to do this - <a href="Link goes here">Text to go to page</a>
