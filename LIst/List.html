What is list?

HTML lists allow web developers to group a set of related items in lists.An unordered list starts with the "ul" tag. Each list item starts with the "li" tag.An ordered list starts with the "ol" tag. Each list item starts with the "li" tag.The list items will be marked with numbers by default: The list items will be marked with bullets (small black circles) by default:

Example -

Unordered list -

  • Coffee
  • Tea
  • Milk

Ordered list -

  1. Coffee
  2. Tea
  3. Milk

code
